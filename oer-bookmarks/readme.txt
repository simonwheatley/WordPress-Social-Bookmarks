== Description ==

Provides a bookmark post type, along with a bookmarklet for adding bookmarks.

== Installation ==

1. Upload the `oer-bookmarks` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to "Tools" in the WordPress admin area and drag "Bookmark This" to your bookmark bar
